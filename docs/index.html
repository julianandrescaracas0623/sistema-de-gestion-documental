Sistema Web de Gestión Documental

Plataforma para digitalizar, organizar y consultar la documentación administrativa de la IPS Salud Integral (Cartago, Valle del Cauca).

Acceder al sistema Ver código en GitHub

Problema

La gestión manual de documentos físicos dificultaba la búsqueda, el control y la conservación de la información institucional.

  • Pérdida de tiempo en consultas y recuperación de archivos.
  • Riesgo de deterioro o pérdida de documentación en papel.
  • Falta de control sobre el acceso a la información.

Solución

Sistema web con autenticación, roles y permisos (RBAC), repositorio centralizado y almacenamiento seguro en la nube.

  • Carga, consulta y descarga de documentos.
  • Categorías, etiquetas y búsqueda avanzada.
  • Administración de usuarios, roles y permisos.
